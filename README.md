[Decision Tree vs. Multilayer Perceptron Research Paper](https://github.com/MilesAnderson/ml-calorie-regression.git)

Implemented Decision Tree and Multilayer Perceptron regression models
in Python to predict calorie expenditure using wearable sensor and
demographic features. Performed extensive hyperparameter tuning with
GridSearchCV and 5-fold cross-validation, evaluated models using MAE
and RMSLE, and analyzed learning curves, feature importance, and
training/inference time on a 750k-sample dataset.
